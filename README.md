
    Pax Dei Latam - Next.js Minimal Portal


    How to deploy:
1. Create Supabase project and run supabase_schema.sql
2. In Supabase Settings -> API get Project URL and ANON key
3. In Vercel, import this repo, set env vars:
   NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY
4. Deploy on Vercel
